This sample shows the 4 of the most commonly used extensions, available to be set as true or false for convenience. You can set the following: commenting: true -> You can comment on items which can be commented on in Connections, such as status updates. saving: true -> You can save/ favourite items. refreshButton: true -> An icon in the upper right corner will allow you to refresh the stream, deleteButton: true -> This will allow deletable items to be deleted.
