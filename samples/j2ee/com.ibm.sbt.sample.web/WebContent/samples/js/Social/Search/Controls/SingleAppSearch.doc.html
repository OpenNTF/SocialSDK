Single App Search sample shows how to search a specific part of connections, for example profiles, communities, files etc. The user can then add a result to a list . add the application parameter to the searchBox passing it the value of the part of connnections you want to search. For example, var searchBox = new SearchBox({ application: "communities" }); To enable adding results to a list use the memberList parameter. var searchBox = new SearchBox({ application: "communities", memberList: true });
