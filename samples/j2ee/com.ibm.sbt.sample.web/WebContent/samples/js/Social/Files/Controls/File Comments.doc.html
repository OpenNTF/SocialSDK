This sample shows a grid of comments for a file written by a speficied user. To set the file grid to display comments use the type parameter with "myFileComments" type: "myFileComments" A userId and documentId parameter must also be set. documentId is the file you wish to view comments for and the userId is the id of the user who made the comments. This sample uses the file service to retrieve the id of a file and the id of the person who owns the file and passes these id's to the fileGrid.
