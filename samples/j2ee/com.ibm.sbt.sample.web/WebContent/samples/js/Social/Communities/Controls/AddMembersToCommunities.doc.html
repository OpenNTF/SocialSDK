Sample showing how to add a one or more selected users to one or more selected communities. The sample shows a public communities grid and a grid displaying a specified users colleagues. You can check the checkbox on each grid row for both grids, then press the add button, this will add The users selected from the profiles grid to the communities selected in the communities grid. This sample shows how to use the bootstrap theme, how to get the selected rows of a grid and how to add users to a community.
