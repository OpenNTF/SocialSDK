This sample shows how to create, update or delete a Community. It will load a Community for which the logged on user is an owner. Enter a valid Community id and clock the Load button to load a different Community. Enter a unique title and click the Create button to create a new Community. Edit the title, content and/or tags and click the Update button to update the currently loaded Community. Click the Delete button to remove the currently loaded Community (Warning: You will not be prompted to confirm this operation).
