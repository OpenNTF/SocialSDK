This sample displays a grid of forums that are owned by the authenticated user. The type parameter when set to "my" will show the forums belonging to the user, when set to public it will show public forums. the baseProfilesUrl, is if you need to change the base url of where profiles are installed. Normally profiles are in [server-Root]/profiles, but if your connections server is installed differnetly you can change this with this paramter. this is for viewing profiles of people in the forum.
