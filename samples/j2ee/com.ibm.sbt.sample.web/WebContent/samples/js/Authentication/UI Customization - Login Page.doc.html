This sample shows how login UI mode and form can be customized. login UI mode is "mainWindow" by default which means your main window will redirect to login page. This behaviour can be changed by changing value of 'loginUi' property as shown in the sample. Similarly User can also have a custom html for popup/mainWindow login ui mode which is "authenticator/templates/login.html" by default. It can be changed by changing value of 'loginPage' property as shown in the sample
